Playit.gg Setup

Follow this guide to get a public IP for your server without manual port forwarding.

1

Download the Plugin

Download the latest .jar file for Bukkit/Spigot/Paper servers from the official Playit.gg repository.

Download playit-minecraft.jar
2

Upload to Cube Hosting

Navigate to your File Manager on the Cube Panel, open the plugins folder, and upload the jar file you just downloaded.

Directory: /home/container/plugins/
3

Generate Link Code

Restart your server. Once the server is online, watch the Console. You will see a message from Playit.gg with a unique claim link.

[playit.gg] To claim your secret, visit: https://playit.gg/claim/XXXX-XXXX
4

Configure the Tunnel

Click the link in your console and log into your Playit.gg account. Follow the on-screen prompts:

  • Select Minecraft Java (or Bedrock if applicable).
  • Ensure the local port is set to 25565 (standard).
  • Click Add Tunnel.
5

Get Your New IP

Once the tunnel is active, Playit.gg will provide you with a custom address (e.g., random-name.playit.gg). Share this address with your friends to allow them to join!

Note: You do NOT need to change your server port in server.properties. Keep it as the default assigned by Cube Hosting.
