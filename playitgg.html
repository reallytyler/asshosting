Playit.gg Setup

Learn how to port forward your Minecraft server on Asshosting using Playit.gg tunnels.

  Playit.gg is the best way to bypass CGNAT or avoid sharing your server's home/node IP directly.
1

Download the Plugin

First, you need the official Playit.gg Bukkit/Spigot plugin. This allows the tunnel to run directly inside your Minecraft process.

  Visit Playit Download Page

Select the Minecraft Plugin (.jar) option.

2

Upload to Asshosting Panel

Go to your server dashboard and follow these steps:

  • Stop your server.
  • Click on the Files tab.
  • Navigate to the /plugins folder.
  • Upload the playit-minecraft-xxx.jar file you just downloaded.
  • Start your server.
3

Linking your Account

Once the server starts, look at the Console. You will see a message from Playit.gg with a unique link.

[playit.gg] Visit this link to link your server:
https://playit.gg/claim/xxxx-xxxx-xxxx

Copy that link into your browser. If you don't see it, type playit setup in the console.

4

Creating the Tunnel

After clicking the link and logging in:

1. Click Add Tunnel.
2. Select Minecraft Java (or Bedrock if applicable).
3. Set the local port to 25565.
4. Click Create Tunnel.

Success! Your server is now online. You will be given a custom IP like random-words.playit.gg. Give this to your friends!
